NAFAS GUNUNG CUPU – Website Statis
================================

Deploy tercepat (Netlify):
1) Netlify → Add new site → Deploy manually
2) Upload folder ini / ZIP
3) Dapat URL (xxxx.netlify.app)
4) Buat QR dari URL final
5) (Opsional) pasang domain pendek: nafascupu.id

Unduhan:
- Letakkan PDF/LKS/KMZ ke folder downloads/
- Ganti nama link di section #unduhan pada index.html

QR papan:
- Ukuran QR: 4–6 cm (A3), 8–12 cm (A1)
- Tambahkan URL teks di bawah QR sebagai cadangan.
